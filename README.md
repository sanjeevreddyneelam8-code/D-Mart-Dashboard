# Dmart Sales Dashboard (Excel)

## Project Overview

The Dmart Sales Dashboard is an interactive Excel dashboard created to analyze and visualize sales data effectively. The dashboard helps in understanding business performance, customer behavior, profit trends, and product analysis using charts, pivot tables, and slicers.

This project is useful for beginners in Data Analysis and Business Intelligence to understand how Excel dashboards are built using real-world sales data.

---

## Objectives

* Analyze overall sales and profit
* Identify top-performing products and categories
* Compare regional sales performance
* Understand customer segments
* Track yearly sales growth
* Improve business decision-making using data visualization

---

## Features

### Sales Analysis

* Total Sales
* Total Profit
* Total Quantity Sold

### Category Analysis

* Furniture
* Office Supplies
* Technology

### Regional Analysis

* State-wise sales comparison
* Region-wise profit analysis

### Customer Insights

* Consumer Segment
* Corporate Segment
* Home Office Segment

### Trend Analysis

* Monthly/Yearly sales trends
* Profit growth over time

### Interactive Elements

* Slicers
* Filters
* Pivot Charts
* Dynamic Reports

---

## Tools & Technologies Used

* Microsoft Excel

* Pivot Charts

* Slicers & Filters

## Dataset Used

The dashboard is created using a retail/superstore sales dataset containing:

* Order Details
* Customer Information
* Product Categories
* Sales & Profit Data
* Shipping Details
* Regional Information
